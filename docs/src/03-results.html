4.2 3.1 M1: 基准回归 — 优序融资 vs 权衡理论

4.2.1 3.1.1 模型设定

\[Lev_{it} = \alpha + \beta \cdot NPR_{it} + \gamma' X_{it} + \mu_i + \lambda_t + \varepsilon_{it}\]

其中： - \(X_{it}\) 为控制变量（Size, Tang, Growth, NDTS） - \(\mu_i\) 为公司固定效应 - \(\lambda_t\) 为年度固定效应 - 标准误在公司与年度层面双向聚类

4.2.2 3.1.2 两步估计法（TWFE）与交互固定效应（IFE）

Note模型对比
  • M1 TWFE：传统双向固定效应，控制公司与年度固定效应
  • M1’ IFE：交互固定效应，允许公司层面系数随时间变化（Cerulli & Dong, 2022）
模型 估计方法 NPR 系数 标准误 t值
M1 TWFE reghdfe -0.618* 0.054 -11.41
M1’ IFE regife -0.392* 0.049 -8.00
变化 缩幅36% -0.226 — —

4.2.3 3.1.3 结果解读

NPR 系数为负（-0.618）：A 股整体支持优序融资理论

  • 经济含义：净利润率每提高1个百分点，资产负债率下降约0.62个百分点
  • 理论含义：盈利越强的企业越倾向于使用内部资金，降低对外源融资的依赖

IFE 压缩但不消除 NPR 效应

  • IFE 系数缩至 -0.392（减少36%）
  • 剩余 64% 反映因果效应，36% 由时变异质性驱动

4.2.4 3.1.4 控制变量结果

变量 系数 标准误 显著性
NPR -0.618 0.054 ***
Size 0.096 0.004 ***
Tang 0.089 0.018 ***
Growth 0.033 0.009 ***
NDTS 0.573 0.164 ***
Tip控制变量解读
  • Size 正向：大规模企业杠杆率更高，支持权衡理论
  • Tang 正向：有形资产越多，抵押能力越强，债务融资更多
  • Growth 正向：高成长企业融资需求更大
  • NDTS 正向：非债务税盾与债务税盾存在替代关系

4.3 3.2 M2: 分组回归 — SOE vs Non-SOE

4.3.1 3.2.1 分组回归结果

分组 NPR 系数 标准误 t值 N
SOE（国企） -0.836* 0.081 -10.32 14,221
Non-SOE（民企） -0.513* 0.069 -7.43 23,017
差异 -0.323 — F=18.65 p=0.0007

4.3.2 3.2.2 产权性质差异的经济解释

国企 NPR 系数更负（-0.836 vs -0.513）：

  1. 预算软约束假说：国企盈利后更倾向于降低杠杆以维持财务稳健，避免政府救助预期
  2. 政治成本假说：国企管理者盈利后主动降杠杆，减少被问责风险
  3. 融资优势转化：国企低融资成本优势未被用于加杠杆，而是储备现金
Important核心发现

国有企业的优序融资效应是民营企业的 1.63 倍（|−0.836|/|−0.513|）

4.5 3.4 M4: 时变系数模型

4.5.1 3.4.1 模型设定

允许 NPR 系数随时间变化：

\[Lev_{it} = \alpha + \beta_t NPR_{it} + \gamma' X_{it} + \mu_i + \lambda_t + \varepsilon_{it}\]

4.5.2 3.4.2 \(\beta_t\) 时序演变

时期 年份区间 \(\beta_t\) 均值 主导理论
第一阶段 2011-2013 -0.74 强优序融资
第二阶段 2015-2020 -0.38 效应减弱
第三阶段 2022-2025 -0.78 优序融资回归

4.5.3 3.4.3 宏观事件影响

时期 重大事件 \(\beta_t\) 变化 机制
2011-2013 四万亿刺激退出 -0.74 → -0.68 融资收紧，优序效应强
2015-2016 去杠杆政策 -0.52 → -0.38 政策主导，融资便利
2018-2019 贸易摩擦 -0.45 → -0.48 风险规避，债务收缩
2022-2025 后疫情复苏 -0.62 → -0.95 现金为王，优序强化
Tip核心结论

NPR-Lev 关系呈现”弱→强→弱→强”的三阶段演变，政策与宏观环境是关键驱动力。

4.6 3.5 M5: 函数系数模型（多项式调节）

4.6.1 3.5.1 模型设定

\[Lev_{it} = \alpha + \beta_0 NPR_{it} + \beta_1 NPR_{it} \times Size_{it} + \beta_2 NPR_{it} \times Size_{it}^2 + \gamma' X_{it} + \mu_i + \lambda_t + \varepsilon_{it}\]

4.6.2 3.5.2 回归结果

变量 系数 标准误 t值
NPR（基准） -30.289* 5.004 -6.05
NPR × Size +2.699* 0.445 +6.06
NPR × Size² -0.061* 0.010 -6.17

4.6.3 3.5.3 边际效应函数

\[\hat{\beta}(Size) = -30.289 + 2.699 \times Size - 0.061 \times Size^2\]

拐点计算：令 \(\hat{\beta}(Size) = 0\)，解得： \[Size^* = \frac{-b_1}{2b_2} = \frac{30.289}{2 \times 0.061} \approx 22.12\]

Size 分位 Size 值 \(\hat{\beta}(Size)\) 符号
P10 20.79 -0.67 负
P25 21.33 -0.38 负
P50 22.06 -0.03 接近零
P75 23.01 +0.49 正
P90 24.07 +0.93 正
Note非线性调节

规模对 NPR-Lev 关系具有显著的倒U型调节效应： - 小规模企业（P10-P25）：NPR 负向效应显著，优序融资主导 - 中等规模（P50附近）：效应趋近于零 - 大规模企业（P75-P90）：NPR 正向效应，权衡理论主导

4.7 3.6 M6: 门槛模型

4.7.1 3.6.1 模型设定

\[Lev_{it} = \alpha + \beta_1 NPR_{it} \cdot I(Size \leq \gamma) + \beta_2 NPR_{it} \cdot I(Size > \gamma) + \gamma' X_{it} + \mu_i + \lambda_t + \varepsilon_{it}\]

门槛值 \(\gamma\) 通过网格搜索最小化残差平方和（SSR）估计。

4.7.2 3.6.2 门槛估计结果

参数 估计值 标准误 t值
\(\hat{\gamma}\)（门槛值） 23.36 0.84 11.39
门槛分位数 ~P64 — —
NPR（Size ≤ \(\gamma\)） -0.666* 0.075 -8.83
NPR（Size > \(\gamma\)） -1.028* 0.097 -10.59
Important意外发现

大规模企业（Size > 23.36）的 NPR 负向效应更强（-1.028 vs -0.666），与 M5 理论预测相反！

可能的解释： 1. 中国大型国企的特殊行为模式：盈利后主动降杠杆以规避风险 2. 大型企业代理问题：管理层过度自信导致的债务扩张被 NPR 负向系数证伪 3. 融资便利下的逆向选择：低融资成本并未导致高杠杆

4.7.3 3.6.3 M5 vs M6 对比

模型 方法 结构性转折位置 发现
M5 多项式（连续） 拐点 ≈ 22.12 P50附近，理论倒U
M6 门槛（离散） \(\gamma\) = 23.36 P64，两区域均为负

两个模型的结构性转折位置接近（22.12 vs 23.36），互为稳健性检验。

4.9 3.8 小结

Note核心结论
  1. 优序融资理论得到支持：NPR 系数在所有模型中为负（-0.618 至 -1.028）
  2. 产权性质是关键的调节变量：国企 NPR 效应显著强于民企
  3. NPR-Lev 关系随时间演变：受政策与宏观环境影响显著
  4. 规模存在非线性调节：但门槛模型发现大规模企业效应反而更强
  5. IFE 压缩但不消除 NPR 效应：因果效应约占64%
