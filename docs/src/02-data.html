3.2 2.1 数据来源

本研究使用 CSMAR（国泰安数据库） 提供的中国A股上市公司数据：

  • 时间跨度：2010—2025年（用于筛选，回归分析使用2011—2025年）
  • 样本范围：全部A股上市公司
  • 数据下载时间：2026年4月22日

3.3 2.2 变量定义

3.3.1 2.2.1 被解释变量

Lev（资产负债率） = 总负债 / 总资产

Note测量时点

使用年末数据，计算方式：\(\text{Lev}_t = \frac{\text{Total Debt}_{it}}{\text{Total Assets}_{it}}\)

3.3.2 2.2.2 核心解释变量

NPR（净利润率） = 净利润 / 总资产

  • 反映企业盈利能力
  • 按资产规模标准化，消除了规模效应

Size（企业规模） = ln（总资产）

  • 取对数处理处理右偏分布
  • 单位：百万元人民币

Tang（有形资产率） = 有形资产 / 总资产

  • 有形资产 = 固定资产 + 在建工程 + 无形资产
  • 反映抵押品价值，影响债务融资能力

Growth（营业收入增长率） = (当年营收 - 上年营收) / 上年营收

  • 衡量企业成长性
  • 高增长企业可能面临更大的融资需求

NDTS（非债务税盾） = 固定资产折旧 / 总资产

  • 作为债务税盾的替代变量
  • 控制税收因素的干扰

3.3.3 2.2.3 产权性质变量

SOE（国有企业虚拟变量）

  • SOE = 1：最终控制人为国有企业或政府机构
  • SOE = 0：民营或外资控股
Tip判断标准

基于CSMAR “最终控制人类型” 字段，政府直接或间接持股超过一定比例则定义为国有企业。

3.3.4 2.2.4 变量汇总表

变量符号 变量名称 计算公式 预期方向
Lev 资产负债率 总负债 / 总资产 被解释变量
NPR 净利润率 净利润 / 总资产 -（优序融资）
Size 企业规模 ln(总资产) +（权衡理论）
Tang 有形资产率 有形资产 / 总资产 +（担保能力）
Growth 营业收入增长率 ΔRevenue / Revenue_{t-1} 不确定
NDTS 非债务税盾 折旧 / 总资产 -（替代效应）
SOE 产权性质 国企=1，民企=0 调节变量

3.4 2.3 样本筛选流程

3.4.1 2.3.1 筛选步骤

3.4.2 2.3.2 筛选标准说明

  1. 金融保险行业排除：金融类企业资产负债率天然较高（>90%），且资本结构决策逻辑不同
  2. ST/PT 公司排除：财务异常公司资本结构处于特殊状态，可能扭曲实证结果
  3. 资不抵债排除：Lev > 1 意味着净资产为负，不符合正常经营假设
  4. 缺失值处理：删除任一关键变量缺失的观测

3.5.3 2.4.3 相关性分析

主要变量间相关系数矩阵显示：

  • Lev 与 NPR 负相关（r = -0.12）：初步支持优序融资理论
  • Lev 与 Size 正相关（r = 0.35）：大规模企业杠杆率更高
  • Lev 与 Tang 正相关（r = 0.24）：有形资产越多，债务融资能力越强
  • NPR 与 Size 正相关（r = 0.10）：大企业盈利能力略强

3.6 2.5 数据处理说明

3.6.1 2.5.1 Winsorize 处理

为控制极端值影响，对所有连续变量在1%和99%分位数进行缩尾处理：

Winsorize(Var, lower=1%, upper=99%)
Tip处理效果

Winsorize 有效降低了异常值对回归结果的杠杆效应，同时保留了数据的分布特征。

3.6.2 2.5.2 平衡面板 vs 非平衡面板

  • M1-M5：使用非平衡面板，保留所有观测值（37,238 obs）
  • M6（门槛模型）：由于估计方法要求，构建平衡面板（7,590 obs, 506 firms, 15 years）

3.7 2.6 小结

Note本章要点
  1. 最终样本：4,513家公司，37,238个观测值，2011-2025年
  2. 核心变量已按学术标准定义，并进行 Winsorize 处理
  3. SOE 与 Non-SOE 在资本结构与盈利能力上存在显著差异
  4. 相关性分析初步支持优序融资理论预测
